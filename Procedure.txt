Files Created:
1. api/app-redirect.js
Vercel Serverless Function that replaces the old PHP redirect flow
Detects Android, iOS, and other devices
Redirects users to the appropriate app store or website

2. vercel.json
Vercel rewrite configuration
Maps old PHP-style paths to the Vercel API route
Keeps backward compatibility for existing links and QR codes

3. download.html
Landing page with manual download buttons for Android and iOS
Includes QR code instructions

How to Deploy:
1. Push all files to your Git repository
2. Import/deploy the repository on Vercel
3. Vercel will pick up `vercel.json` and `api/` routes automatically
4. Test the redirect at: https://<your-domain>/app-redirect.php

QR Code Setup:
Generate a QR code pointing to: https://<your-domain>/app-redirect.php

The URL will redirect users to:
- Android users -> Google Play Store
- iOS users -> Apple App Store
- Desktop users -> Your website
